final class KodiHost

The KodiHost class

This class takes care of:

  • Getting the properties from the host
  • Get and set volume
  • Fiddling with ReplayGain
  • Scanning the library on request

ReplayGain

  • setReplayGain(mode:) Asynchronous

    Set Kodi ReplainGain setting

    • When playing from a selected album; ReplayGain will be set to ‘album’ mode so the album is played as intended.

    • When playing random songs; ReplayGain will be set to ‘track’ so all songs are played at the same level.

    Note

    This wil alter the Kodi host settings and when quitting Kodio it will not be set back!

    Declaration

    Swift
