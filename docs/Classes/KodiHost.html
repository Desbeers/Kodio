ReplayGain

  • setReplayGain(mode:)

    Set Kodi ReplainGain setting

    When playing from a selected album; ReplayGain will be set to ‘album’ mode so the album is played as intended. When playing random songs; ReplayGain will be set to ‘track’ so all songs are played at the same level.
